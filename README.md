# Myntra-Sales-Analysis
# Objective:
This project analyzes a Myntra sales dataset to uncover key business and product insights across brands, pricing, ratings, and discounts. It provides exploratory data analysis (EDA), visualizations, and metrics useful for merchandising, pricing strategy, inventory decisions, and understanding customer preferences.

# Key Features of Dataset:
- brand_name	Clothing brand (e.g., WROGN, Levi's, Roadster)
- pants_description	Product title / jeans description
- price	Current selling price (after discount)
- MRP	Original price before discount
- discount_percent	Discount applied (calculated as (MRP - price) / MRP)
- ratings	Average user rating
- number_of_ratings	Count of reviewers / ratings received
